# War Visualization (Skill Distillery Project, Week 2)
#### By Joshua Bone, 10/10/2016



### Overview
This is a text-based visualization of the card game 'War', with a twist. It shows the balance of power winding back and forth as two computer players play the game. See sample output below.

##### Notes
Begin the game by running the War class.
